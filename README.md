# JOYA 20 ELITE — Versión 0.1

Primera base modular de JOYA 20.

## Incluye

- API-Football Pro.
- Scanner automático de una cantidad configurable de partidos.
- Filtros iniciales para competiciones volátiles.
- JOYA Score inicial.
- Ranking diario.
- Mercados adaptados al nombre habitual de Betano.
- Constructor de cartillas por:
  - cuota objetivo;
  - máximo de picks;
  - Tier mínimo.
- Disciplina de no forzar picks cuando no se alcanza el objetivo.

## Importante

La cuota combinada de esta versión se calcula con una cuota media ingresada manualmente.
Todavía no consulta cuotas reales de Betano.

## Streamlit Secrets

```toml
APISPORTS_KEY = "TU_CLAVE"
```

## Instalación recomendada

Crea un repositorio nuevo llamado `JOYA-20-ELITE` y sube todo el contenido
de esta carpeta conservando las subcarpetas.
