# JOYA 20 ELITE — v0.2 Ligas

## Cambios principales

- Revisa todas las ligas disponibles del día.
- Separa la cartelera por país y liga.
- Distribuye el scanner entre todas las ligas.
- Permite elegir cuántos partidos analizar por cada liga.
- Resultados agrupados por liga.
- Selector de ligas disponibles en Betano.
- Opción de máximo un pick por liga.
- Constructor de cartillas con ligas seleccionadas.

## Instalación

Reemplaza el contenido del repositorio JOYA 20 con todos estos archivos y carpetas,
manteniendo exactamente los nombres:

- app.py
- requirements.txt
- api/
- config/
- engine/
- ui/

Mantén en Streamlit Secrets:

```toml
APISPORTS_KEY = "TU_CLAVE"
```
