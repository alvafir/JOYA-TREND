# JOYA 22 AI v1

## Incluye

- Score calibrado para evitar exceso de 98-100.
- Nueva escala:
  - 95-99 S++
  - 90-94 S+
  - 85-89 A++
  - 80-84 A+
  - menos de 80 NO BET
- Explicación por pick.
- Radar visual.
- Tendencias local en casa y visitante fuera.
- Tendencias por minutos.
- Scanner con resumen del proceso.
- Constructor inteligente:
  - máximo por liga;
  - máximo por país;
  - máximo del mismo mercado;
  - máximo del mismo grupo.
- JOYA Brain mediante CSV histórico.

## Plantilla JOYA Brain

La app permite descargar una plantilla con columnas:

- Liga
- Mercado
- Picks
- Aciertos
