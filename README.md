# JOYA 19 PRO — Fase 1

## Archivos que debes subir a GitHub

- `app.py`
- `requirements.txt`

## Streamlit Secrets

En la configuración de tu aplicación, abre **Secrets** y guarda:

```toml
APISPORTS_KEY = "TU_CLAVE_NUEVA"
```

No pongas la clave dentro de `app.py`.

## Funciones incluidas

- Conexión a API-Football Pro.
- Estado y consumo diario de solicitudes.
- Partidos por fecha.
- Filtro por país, liga y equipo.
- Últimos 10 partidos.
- H2H.
- Clasificación.
- Predicciones de API-Football.
- Lesiones.
- Cuotas prepartido, según cobertura.

La cobertura de estadísticas, lesiones, predicciones y cuotas puede variar por competición.
