# JOYA 19 PRO — Fase 2 inicial

## Instalación

Reemplaza en GitHub:

- `app.py`
- `requirements.txt`

Mantén en Streamlit Secrets:

```toml
APISPORTS_KEY = "TU_CLAVE"
```

## Incluye

- Partidos por fecha.
- Filtros por país, liga y equipo.
- Análisis de los últimos 10 partidos.
- JOYA Score inicial.
- Ranking de mercados.
- Tier interno S++, S+, A++ o NO BET.
- Mercados:
  - Más de 1.5 goles
  - Menos de 4.5 goles
  - Ambos anotan
  - Local marca
  - Visitante marca
  - 1X
  - X2

Los porcentajes son tendencias descriptivas, no garantías.
