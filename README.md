# JOYA 20 ELITE v0.6 — Todos los mercados

Esta versión mantiene todo lo anterior e integra el Goal Engine como opciones nuevas.

## Incluye

- Scanner por todas las ligas.
- Resultados separados por país y liga.
- Mercados de goles.
- BTTS.
- Doble oportunidad.
- Primer tiempo.
- Minutos:
  - ningún gol antes del 10;
  - gol antes del 70;
  - doble condición 0-10 + antes del 70;
  - local/visitante marca antes del 70.
- Primer gol:
  - local marca primero;
  - visitante marca primero.
- Constructor de cartillas.
- Próximo gol en vivo.

## Instalación

Reemplaza únicamente:

- app.py
- requirements.txt
- README.md

Mantén en Streamlit Secrets:

```toml
APISPORTS_KEY = "TU_CLAVE"
```
